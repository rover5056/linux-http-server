# 欢迎使用Linux-http-server
Linux上将静态文件目录变成服务器的小工具
## 安装

```
npm i linux-http-server -g
```

## 启动


```
rover
```
>  会在当前目录下开设端口，默认index.html为首页，也可自定义

---



## 参数

```
rover -p  3000               //设置端口为3000
rover -d  /usr               //设置 /usr 目录为根目录
rover -f /usr/local/         //启动目录为/usr/local/
```
